# 3.5 多处理机系统

> 考试定位：综合知识（选择题） | 预计分值：2-3 分 | 出题频率：中

---

## 一、核心概念

### 1.1 多处理机基本概念

多处理机（Multi Processor）具有两个或两个以上处理机，共享I/O子系统，在OS统一控制下通过共享主存或高速通信网络协同求解复杂问题。属于MIMD（Multiple Instruction Multiple Data，多指令多数据流）结构。

### 1.2 多处理机 vs 并行处理机

并行处理机基于SIMD（Single Instruction Multiple Data，单指令多数据流）结构，一个控制器控制多个处理单元做相同运算，适合矩阵/向量计算（又称矩阵处理机、向量处理机或阵列处理机）。两者从结构灵活性、程序并行性、并行任务派生、进程同步、资源分配/调度5个方面存在差异。

### 1.3 紧耦合系统 vs 松耦合系统

紧耦合系统（共享存储方式）：处理机共享公共存储器SM，如SMP（Symmetric Multi-Processor，对称多处理机）。特点是编程容易但扩展困难。

松耦合系统（分布式存储方式）：每个处理机独占本地存储器LM，如MPP（Massively Parallel Processor，大规模并行处理机）。特点是容易扩展但编程困难，适合粗粒度并行。

### 1.4 SVM（共享虚拟存储器）

SVM（Shared Virtual Memory）在分布存储的多处理机上实现物理分布、逻辑共享的存储系统。实现途径：硬件实现、操作系统和库实现、编译实现。

### 1.5 共享存储的三种模型

1. **UMA**（Uniform Memory Access，均匀存储器存取）：物理存储器被所有处理机均匀共享。
2. **NUMA**（Nonuniform Memory Access，非均匀存储器存取）：访问时间随存储字位置变化，本地快、远程慢。
3. **COMA**（Cache Only Memory Architecture，只用高速缓存的存储器结构）：NUMA特例，分布主存换成Cache。

### 1.6 S2MP（可扩展共享存储多处理机）

S2MP（Scalable Shared Memory Multi-Processing）是一种NUMA结构，每个结点由处理机和存储器组成，支持简单编程模型，兼顾性能和扩展性。

### 1.7 互连网络

互连网络连接各处理部件、存储模块及外设。基本互连函数：恒等置换、交换置换、方体置换、均匀洗牌置换（Shuffle）、蝶式置换、位序颠倒置换。

多处理机互连方式（5种）：总线方式（最简单，争用最严重）、交叉开关（争用最少，复杂度最高）、开关枢纽（仲裁+开关）、多端口存储器、多级互连网络（总线与交叉开关的折中）。

## 三、概念对比

| 概念 | 核心区别 | 记忆技巧 |
|---|---|---|
| SIMD vs MIMD | SIMD:多个处理单元执行相同指令、处理不同数据（阵列机）；MIMD:多个处理机执行不同指令、处理不同数据（多处理机） | SIMD=军队齐步走，MIMD=各干各的 |
| 紧耦合(SMP) vs 松耦合(MPP) | 紧耦合:共享SM、编程易但扩展难；松耦合:分布式LM、易扩展但编程难 | 紧=大家庭共住，松=邻居分别住 |
| UMA vs NUMA | UMA:均匀访问时间；NUMA:访问本地快、远程慢 | U=均匀，N=不均匀 |
| 总线 vs 交叉开关 | 总线:最简单、争用最严重；交叉开关:最复杂、争用最少 | 总线=一条水渠共用，交叉开关=每家有专用管道 |
| 并行处理机 vs 多处理机 | 并行:操作级并行(SIMD)；多处理:任务级并行(MIMD、多程序段并发) | 并行=干同一件事，多处理=各干各事 |

## 四、历年真题/模拟题

### 模拟题1

**题目：** 在Flynn分类法中，并行处理机基于（ ），多处理机基于（ ）。

A. SISD, SIMD
B. SIMD, MIMD
C. MISD, MIMD
D. MIMD, SIMD

**答案：** B

**解析：** 并行处理机属于SIMD（单指令多数据流），一个控制器控制多个处理单元做同样运算；多处理机属于MIMD（多指令多数据流），各处理机独立执行不同指令。

---

### 模拟题2

**题目：** 以下关于SMP的说法，正确的是（ ）。

A. SMP属于松耦合系统
B. SMP各处理机独占本地存储器
C. SMP属于紧耦合系统，处理机共享公共存储器
D. SMP采用分布式存储方式

**答案：** C

**解析：** SMP（对称多处理机）属于紧耦合系统，处理机共享公共SM。MPP属于松耦合系统，各处理机独占LM。

---

### 模拟题3

**题目：** 在共享存储多处理机中，如果访问本地存储器很快，访问远程存储器比较慢，这种模型是（ ）。

A. UMA
B. NUMA
C. COMA
D. SVM

**答案：** B

**解析：** NUMA（非均匀存储器存取）中，处理机访问LM快，访问远程存储器需通过互连网络产生附加时延。

---

### 模拟题4

**题目：** 以下互连方式中，连接复杂度最低但争用最严重的是（ ）。

A. 总线方式
B. 交叉开关
C. 多级互连网络
D. 多端口存储器

**答案：** A

**解析：** 总线最简单、连接复杂度最低，但共享总线导致争用最严重。交叉开关争用最少但复杂度最高。

## 五、背诵知识点

1. 【定义】并行处理机=SIMD，多处理机=MIMD。SIMD单指令多数据流，MIMD多指令多数据流
2. 【对比】紧耦合(SMP)→共享SM→易编程难扩展；松耦合(MPP)→分布式LM→易扩展难编程
3. 【口诀】共享存储3模型：UMA(均匀快)、NUMA(不均匀本地快远程慢)、COMA(全Cache)
4. 【对比】5种互连方式：总线最简单(争用多)→多级网络→交叉开关最复杂(争用少)
5. 【口诀】互连函数6种：恒等=不变、交换=最低位反、方体=第k位反、Shuffle=循环左移、蝶式=头尾换、位序颠倒=全倒序
6. 【定义】多处理机与并行处理机5大差异：结构灵活性→程序并行性→并行任务派生→进程同步→资源分配
