# 3.1 计算机系统概述

> 考试定位：综合知识（选择题） | 预计分值：2-3 分 | 出题频率：中

---

## 一、核心概念

### 1.1 计算机系统的多级层次结构

计算机系统可以按功能抽象为自底向上的三级层次：

1. **硬件层（裸机）**：最底层，可细分为硬联逻辑级→微程序级→传统机器级。硬联逻辑级由门电路和触发器组成；微程序级的机器语言是微指令集；传统机器级的机器语言是该机的指令集（即程序员面对的"裸机"界面）。
2. **系统层**：中间层，包含操作系统级和语言处理程序级（编译程序、汇编程序、解释程序）。操作系统管理软硬件资源，语言处理程序将高级/汇编语言翻译为机器语言。
3. **应用层**：最上层，面向普通用户的各类应用软件。

> 记忆口诀：**"硬—系—应"，自底向上越来越"软"、离用户越来越近。**

### 1.2 冯·诺依曼体系结构

现代计算机的基础架构，核心思想是**存储程序原理**。由5大基本部件组成：

1. **运算器（ALU，Arithmetic Logic Unit）**：执行算术和逻辑运算
2. **控制器（CU，Control Unit）**：控制指令执行顺序、协调各部件工作
3. **存储器（Memory）**：存储程序和数据（分为内存和外存）
4. **输入设备（Input Device）**：向计算机输入数据和命令
5. **输出设备（Output Device）**：输出计算结果

运算器和控制器集成在一起统称为**中央处理单元（CPU，Central Processing Unit）**。输入设备和输出设备合称为**外部设备（外设）**。

> 关键要点：冯·诺依曼结构的特点是**数据和指令存放在同一存储器中**、**指令按顺序执行**，区别于哈佛结构（程序和数据分开存储）。

### 1.3 计算机软件分类

软件按功能分为两类：

1. **系统软件**：管理系统硬件、服务其他软件的程序。包括5类：
   - 操作系统（OS，Operating System）
   - 语言处理程序（编译程序、汇编程序、解释程序）
   - 服务性程序（如诊断工具、调试工具）
   - 数据库管理系统（DBMS，Database Management System）
   - 计算机网络软件
2. **应用软件**：为特定业务需求开发的程序，如财务管理软件、项目管理软件。

### 1.4 固件（Firmware）

**固件**是存储在永久保存器件（如 EPROM、EEPROM）中的程序，是**具有软件功能的硬件**。关键特征：

- 执行速度：**快于软件，慢于纯硬件**
- 灵活性：**优于纯硬件，低于软件**
- 作用：操作系统通过固件驱动特定设备运转

## 三、概念对比

| 概念 | 核心区别 | 记忆技巧 |
|---|---|---|
| 内存 vs 外存 | 内存速度快容量小（易失），外存速度慢容量大（持久） | "内存是桌面（临时放），外存是书柜（永久存）" |
| 系统软件 vs 应用软件 | 系统软件管电脑自身，应用软件管用户业务 | 系统软件 = "管家"，应用软件 = "客人" |
| 固件 vs 软件 | 固件固化在硬件中，速度更快、不可随意修改 | 固件 = "焊在里的软件" |
| 微程序级 vs 传统机器级 | 微程序级语言是微指令集（硬件内部），传统机器级语言是该机指令集（对外接口） | 微程序 = 内部齿轮，机器指令 = 方向盘 |
| 冯·诺依曼 vs 哈佛结构 | 冯·诺依曼数据和指令共用一根总线/存储器，哈佛结构分开存储 | 冯 = 单行道，哈佛 = 双车道 |

## 四、历年真题/模拟题

### 模拟题1

**题目：** 下列关于计算机多级层次结构的说法中，正确的是（ ）。

A. 硬件层仅包含硬联逻辑级
B. 系统层中的操作系统负责将高级语言程序翻译为机器语言
C. 应用层是面向用户设计的各种应用程序
D. 微程序级的机器语言就是该机的指令集

**答案：** C

**解析：** A错，硬件层还包含微程序级和传统机器级；B错，翻译程序是语言处理程序的工作而非操作系统；D错，传统机器级的机器语言才是该机的指令集，微程序级语言是微指令集。

---

### 模拟题2

**题目：** 冯·诺依曼体系结构中，计算机的5大基本组成部件不包括（ ）。

A. 运算器
B. 控制器
C. 硬盘
D. 输入设备

**答案：** C

**解析：** 5大部件是运算器、控制器、存储器、输入设备、输出设备。硬盘属于存储器（外存）的一种具体实现，不是独立的基本部件。

---

### 模拟题3

**题目：** 下列关于固件的描述，错误的是（ ）。

A. 固件存储在 EPROM 或 EEPROM 中
B. 固件的执行速度快于软件但慢于硬件
C. 固件的灵活性优于硬件但低于软件
D. 固件等同于操作系统

**答案：** D

**解析：** 固件（Firmware）是具有软件功能的硬件，不是操作系统。操作系统需要通过固件来驱动设备。

---

### 模拟题4

**题目：** 系统软件通常不包括（ ）。

A. 操作系统
B. 编译程序
C. 数据库管理系统
D. 财务管理软件

**答案：** D

**解析：** 财务管理软件属于应用软件。系统软件5类：操作系统、语言处理程序、服务性程序、数据库管理系统、计算机网络软件。

---

### 模拟题5

**题目：** 在计算机系统的层次结构中，将高级语言编写的程序翻译为目标机器语言程序的是（ ）。

A. 操作系统
B. 语言处理程序
C. 微程序
D. 固件

**答案：** B

**解析：** 语言处理程序（包括编译程序、汇编程序、解释程序）位于系统层，负责将高级语言或汇编语言程序翻译为机器语言。注意这是系统层的语言处理程序级，不是操作系统级。

## 五、背诵知识点

1. 【定义】计算机系统的3级层次结构：硬件层（裸机）→ 系统层（OS + 语言处理程序）→ 应用层（用户程序）
2. 【定义】冯·诺依曼结构5大部件：运算器 + 控制器 + 存储器 + 输入设备 + 输出设备
3. 【定义】CPU（中央处理单元，Central Processing Unit）= 运算器（ALU）+ 控制器（CU）
4. 【对比】内存：速度快、容量小、临时存放；外存：速度慢、容量大、持久保存
5. 【口诀】硬件层三级：硬联逻辑 → 微程序级（微指令集）→ 传统机器级（指令集）
6. 【对比】系统软件管电脑（OS、编译程序等5类），应用软件管业务（财务、项目管理等）
7. 【定义】固件 = 存储在 EPROM/EEPROM 中具有软件功能的硬件，性能介于软硬件之间
8. 【口诀】冯·诺依曼核心：**"存程序、顺序执行、五大件、CPU统管"**
9. 【对比】输入设备：数据进；输出设备：结果出；合称外设
10. 【定义】语言处理程序三种：编译程序（全篇翻译）、汇编程序（汇编→机器）、解释程序（逐句翻译）
