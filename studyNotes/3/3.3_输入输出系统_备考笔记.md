# 3.3 输入输出系统

> 考试定位：综合知识（选择题） | 预计分值：2-3 分 | 出题频率：中

---

## 一、核心概念

### 1.1 I/O系统的5种工作方式

从低级到高级分为5种：程序控制方式 → 程序中断方式 → DMA工作方式 → 通道方式 → I/O处理机。呈现"CPU参与越来越少，效率越来越高"的趋势。

### 1.2 程序控制方式

由CPU执行I/O程序完成数据传送。分为无条件传送和程序查询方式。无条件传送默认外设始终就绪；程序查询方式中CPU循环查询外设状态直至就绪，严重影响CPU利用率。

### 1.3 程序中断方式

CPU执行程序时遇到紧急事件，暂时中止当前程序（保护现场）→处理中断→返回原程序（恢复现场）。中断过程的5个阶段：中断请求→中断判优→中断响应→中断处理→中断返回。

### 1.4 DMA工作方式

DMA（Direct Memory Access，直接存储器访问）用于主存与外设间的高速批量数据交换。由DMAC（DMA控制器）控制总线。DMAC获取总线的三种方式：CPU停止法（暂停方式）、存储器分时法（共享方式）、周期挪用法（周期窃取方式）。

### 1.5 通道方式

通道是高级I/O控制部件，包括3种类型：字节多路通道（低速设备，字节交叉）、选择通道（高速设备，独占通道）、数组多路通道（折中，子通道分时+成组传输）。

### 1.6 I/O处理机

I/O处理机接近一般处理机结构，可独立异步工作，有自己指令系统和局部存储器，还能完成码制转换、数据校验、故障处理等。

### 1.7 总线

总线是一组多个部件分时共享的公共信息传送线路。按功能分为：地址总线、数据总线、控制总线。主要性能指标：总线宽度、总线带宽（=总线宽度×总线频率）、总线负载、总线分时复用、总线猝发传输。

### 1.8 I/O接口

接口是主机和外设之间的界面。主要功能有5个：通信联络控制、地址译码/设备选择、数据缓冲、数据格式变换、传递控制命令和状态信息。常见接口：ATA、SATA、SCSI、USB、IEEE-1394等。端口编址方式：独立编址（I/O映射）和统一编址（存储器映射）。

## 三、概念对比

| 概念 | 核心区别 | 记忆技巧 |
|---|---|---|
| 程序查询 vs 中断 | 查询:CPU主动循环等待；中断:外设主动通知CPU | 查询=打电话催，中断=发短信通知 |
| 中断 vs DMA | 中断:CPU参与数据传送；DMA:CPU放弃总线，DMAC全权处理 | 中断=CPU自己搬，DMA=助手全包 |
| 字节多路通道 vs 选择通道 vs 数组多路通道 | 字节多路:低速、字节交叉；选择:高速、独占；数组多路:折中、子通道分时+成组 | 字节=共享自行车，选择=独占出租车，数组=拼车 |
| 独立编址 vs 统一编址 | 独立:I/O地址空间和主存地址空间分离，有专门I/O指令；统一:I/O和主存统一编址，用数据传送指令 | 独立=身份证号单独编，统一=全国大编号 |
| 串行接口 vs 并行接口 | 串行:1位/次，远距离；并行:多位/次，近距离 | 串行=单车道，并行=多车道 |

## 四、历年真题/模拟题

### 模拟题1

**题目：** 在以下I/O方式中，CPU参与最少的是（ ）。

A. 程序控制方式
B. 中断控制方式
C. DMA方式
D. 通道方式

**答案：** D

**解析：** 5种方式CPU参与程度从高到低：程序控制>中断>DMA>通道>I/O处理机。通道方式比DMA更独立，CPU仅需发出启动和停止命令。

---

### 模拟题2

**题目：** 某系统总线时钟频率为10MHz，在一个总线周期中并行传输4B信息，一个总线周期占用2个时钟周期，则该总线的带宽为（ ）MB/s。

**答案：** 20MB/s

**解析：** 时钟周期T=1/10M=0.1μs，总线周期=2T=0.2μs，总线带宽=4B/0.2μs=20MB/s。

---

### 模拟题3

**题目：** 关于I/O端口的编址方式，以下说法正确的是（ ）。

A. 独立编址方式中，不需要专门的I/O指令
B. 统一编址方式中，端口和主存使用同一地址空间
C. 独立编址也称为存储器映射
D. 统一编址需要专用的I/O指令

**答案：** B

**解析：** 统一编址（存储器映射）中端口地址和主存单元地址统一编址，不设专门I/O指令。独立编址（I/O映射）需要专门的I/O指令。

---

### 模拟题4

**题目：** DMA控制器获取总线的方式不包括（ ）。

A. CPU停止法
B. 存储器分时法
C. 周期挪用法
D. 中断响应法

**答案：** D

**解析：** DMAC获取总线有3种方式：CPU停止访问主存法（暂停方式）、存储器分时法（共享方式）、周期挪用法（周期窃取方式）。中断响应法不是DMA获取总线的方式。

## 五、背诵知识点

1. 【口诀】I/O方式演进（CPU参与递减）：程序控制→中断→DMA→通道→I/O处理机，记忆为"程中D通I"
2. 【口诀】中断5阶段：请求→判优→响应→处理→返回（请优应处返）
3. 【对比】DMA三方式：暂停（CPU停工）、共享（时间瓜分）、窃取（偷空闲周期）
4. 【对比】通道三类型：字节多路（低速多设备）、选择（高速独占）、数组多路（两者折中）
5. 【公式】总线带宽=总线宽度×总线频率
6. 【定义】总线带宽指标：宽度、带宽、负载、分时复用、猝发传输
7. 【对比】I/O接口5功能：通信联络、地址译码、数据缓冲、格式变换、传递命令状态
8. 【对比】串行通信：异步（需要起止位，效率低）、同步（需要同步时钟，效率高）
