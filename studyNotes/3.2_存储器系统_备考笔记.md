# 3.2 存储器系统

> 考试定位：综合知识（选择题）+ 案例分析 | 预计分值：3-5 分 | 出题频率：高

---

## 一、核心概念

### 1.1 存储器的层次结构

计算机存储器系统采用三级层次结构：高速缓冲存储器（Cache）→ 主存储器（内存）→ 辅助存储器（外存）。越靠近CPU，速度越快、容量越小、价格越高。存储器数据存取方式有4种：顺序存取、直接存取、随机存取和相联存取。性能指标包括存取时间、存储器带宽、存储器周期和数据传输率。

### 1.2 主存储器（Main Memory）

CPU可直接访问。分为RAM（Random Access Memory，随机存取存储器）和ROM（Read Only Memory，只读存储器）。RAM又分为DRAM（Dynamic RAM，需刷新）和SRAM（Static RAM，不需刷新）。

### 1.3 辅助存储器

包括磁带（顺序存取）、硬盘（直接存取）、RAID、光盘。硬盘存取时间Ta≈平均寻道时间Ts+平均等待时间Tw。格式化容量=n×t×s×b。硬盘有三类：HDD（机械）、SSD（固态）和SSHD（混合）。

### 1.4 硬盘存储器

硬盘存取时间 = 寻道时间 + 等待时间 + 读写操作时间。格式化容量公式：存储容量 = n × t × s × b（n=记录面数、t=每面磁道数、s=每道扇区数、b=每扇区字节数）。

### 1.5 磁盘阵列（RAID）

RAID(原为廉价磁盘冗余阵列 Redundant Array of Inexpensive Disks，现为独立磁盘冗余阵列 Redundant Array of Independent Disks)。分8级：RAID 0~7 和 RAID 10。

### 1.6 高速缓冲存储器（Cache）

Cache利用程序局部性原理（时间局部性+空间局部性）突破冯·诺依曼瓶颈。三种地址映射方式：直接映射、全相联映射、组相联映射。替换算法：随机、FIFO、LRU。写策略：写直达（Write Through）、写回（Write Back）、标记法。

### 1.7 网络存储技术

三种技术：DAS（Direct Attached Storage，直接附加存储）、NAS（Network Attached Storage，网络附加存储）、SAN（Storage Area Network，存储区域网络）。SAN又分为FC SAN、IP SAN（iSCSI）和IB SAN。

### 1.8 虚拟存储技术

虚拟存储是将多个物理存储介质统一管理，形成存储池（Storage Pool）。按拓扑分为对称式（嵌入数据路径）和非对称式（独立于数据路径）；按实现原理分为数据块虚拟和虚拟文件系统。实现方式分为主机级、存储设备级和网络级。

## 三、概念对比

| 概念 | 核心区别 | 记忆技巧 |
|---|---|---|
| DRAM vs SRAM | DRAM需刷新，速度慢，价格低；SRAM不需刷新，速度快，价格高 | D=动态(需充电保养)；S=静态(常驻保持) |
| 直接映射 vs 全相联 vs 组相联 | 直接:多对一，简单但不灵活；全相联:任意映射，灵活但太慢/太贵；组相联:折中 | 直接=固定座位，全相联=自由座位，组相联=分组自由坐 |
| 写直达 vs 写回 | 写直达:同时写Cache和内存；写回:只写Cache淘汰时再写内存 | 写直达=当面记账，写回=离职再报 |
| DAS vs NAS vs SAN | DAS:直接连服务器；NAS:网络文件级共享；SAN:网络块级存储 | DAS=一条线，NAS=文件服务，SAN=块级专网 |
| 顺序存取 vs 随机存取 | 顺序:数据按序访问(磁带)；随机:任意位置访问(内存) | 顺序=磁带倒带，随机=任意跳转 |
| RAID 0 vs RAID 1 vs RAID 5 | RAID 0:分块无冗余(利用率100%)；RAID 1:镜像(利用率50%)；RAID 5:分布式校验(利用率n-1/n) | 0=只有性能，1=只有安全，5=两者折中 |

## 四、历年真题/模拟题

### 模拟题1

**题目：** 某计算机主存读/写时间为100ns，有一个指令和数据合一的Cache，该Cache读/写时间为10ns。取指令命中率为98%，取数命中率为95%。约有1/5指令需要存取一个操作数。假设指令流水线任何时候都不阻塞，设置Cache后每条指令平均访存时间约为（ ）。

**答案：** 14.7ns

**解析：** 平均访存时间 = (2%×100ns+98%×10ns) + 1/5×(5%×100ns+95%×10ns) = 2+9.8+1/5×14.5 = 11.8+2.9 = 14.7ns。

### 模拟题2

**题目：** 以下关于RAID技术的描述中，正确的是（ ）。

A. RAID 0具有最高磁盘空间利用率，且具有冗余功能
B. RAID 1由磁盘对组成，磁盘空间利用率为50%，安全性最高
C. RAID 5将校验信息集中存放在一个独立的校验盘上
D. RAID 10建立在RAID 3和RAID 5基础上

**答案：** B

**解析：** A错在RAID 0没有冗余功能；C错在RAID 5没有独立校验盘，校验信息分布；D错在RAID 10建立在RAID 0和RAID 1基础上。

---

### 模拟题3

**题目：** Cache的地址映射中，若主存地址按I mod C的方式直接确定Cache块号，则该映射方式为（ ）。

A. 全相联映射
B. 直接映射
C. 组相联映射
D. 全映射

**答案：** B

**解析：** 直接映射关系为K=I mod C，其中K为Cache块号，I为主存页号，C为Cache块数。该方式硬件简单，但可能浪费部分Cache空间。

---

### 模拟题4

**题目：** 以下网络存储技术中，采用块级别存储的是（ ）。

A. DAS
B. NAS
C. SAN
D. 以上都不是

**答案：** C

**解析：** SAN通过专用交换机构建高速子网，采用块（block）级别存储，不采用文件共享方式。NAS采用文件级共享，DAS直接连接方式。

## 五、背诵知识点

1. 【公式】Cache平均访问时间：t3 = t1×h + t2×(1-h)，命中率h提高即使很小也能大幅改善性能
2. 【对比】RAM（随机存取、可读写、断电丢失）分为DRAM（需刷新、慢、便宜）和SRAM（无需刷新、快、贵）
3. 【对比】直接映射：K=I mod C；全相联：任意映射但速度慢；组相联：J=I mod Q，折中方案
4.【公式】硬盘格式化容量 = n×t×s×b，平均存取时间Ta≈Ts+Tw
5. 【对比】RAID 0无冗余性能最高，RAID 1镜像最安全，RAID 5分布式校验性价比高，RAID 6双校验
6. 【口诀】Cache写策略：写直达（即时同步）、写回（延迟同步）、标记法（无效位标记）
7. 【对比】DAS连服务器距离短难扩展；NAS文件级即插即用；SAN块级专用高速网
8. 【定义】存储虚拟化：将物理存储与逻辑表示分离，形成存储池，实现方式分主机级/存储设备级/网络级
